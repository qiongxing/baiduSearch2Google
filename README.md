
# baiduSearch2Google
谷歌插件，在百度的搜索栏附近增加谷歌搜索

# 插件说明

 - 第一次写谷歌插件,功能上可能出现bug
 - 在进行百度搜索的首页或者搜索栏，附近有谷歌搜索，点击会将搜索栏数据，新开窗口，进行谷歌搜索
